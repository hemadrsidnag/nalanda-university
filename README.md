# Nalanda University Website

This project is a modern, responsive website for Nalanda University, built using Next.js, React, and Tailwind CSS. It aims to provide comprehensive information about the university, its programs, news, and contact details in an accessible and user-friendly format.

## Features

- **Home Page**: Welcoming introduction to the university.
- **About Us**: Detailed information about the university's history, mission, and vision.
- **Programs**: Overview of academic programs offered.
- **News & Events**: Latest updates and events from the university.
- **Contact Us**: Information for inquiries and communication.
- **Responsive Design**: Optimized for various devices (desktop, tablet, mobile).

## Technologies Used

- **Next.js**: React framework for production.
- **React**: A JavaScript library for building user interfaces.
- **Tailwind CSS**: A utility-first CSS framework for rapid UI development.
- **TypeScript**: Superset of JavaScript that adds static types.

## Getting Started

Follow these instructions to set up and run the project locally.

### Prerequisites

Make sure you have Node.js (version 18 or higher) and npm installed on your system.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/nalanda-university.git
   ```
2. Navigate to the project directory:
   ```bash
   cd nalanda-university
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

### Running the Development Server

To start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the application. The page will hot-reload as you make changes.

### Building for Production

To build the application for production:

```bash
npm run build
```

This will create an optimized production build in the `.next` directory.

### Starting the Production Server

To start the production server after building:

```bash
npm run start
```
